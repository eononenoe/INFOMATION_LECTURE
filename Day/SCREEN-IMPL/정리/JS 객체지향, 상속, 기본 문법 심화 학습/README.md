**Prototype(프로토타입)**과 **상속** 개념을 연습합니다.

---

## 💡 핵심 개념 요약 (Key Concepts)

- **객체(Object)**: 데이터와 기능(메소드)를 모아둔 것 📦
- **프로토타입(Prototype)**: 객체 간 상속을 구현하는 핵심 메커니즘
- **배열 함수 활용**: `forEach`, `map`, `filter`, `reduce`
- **상속(Inheritance)**: 부모(상위) 객체의 속성과 기능을 자식 객체가 물려받는 구조 🧬
- **기본 문법(연산자, 조건문, 반복문)**: JS의 근본기를 단단히 다지기

## ⚠ 주의사항 (Cautions)

- 객체를 `freeze()` 하면 **수정 불가**, `seal()` 하면 **새 속성 추가만 불가**.
- Prototype을 사용할 때는 **this** 키워드가 객체 자신을 가리키는지 주의!
- 반복문은 **탈출 조건이 없으면 무한루프**가 될 수 있다! (꼭 탈출 조건 작성)
- 연산자 사용 시 **== vs ===** (동등비교 vs 완전일치비교) 차이 주의!
