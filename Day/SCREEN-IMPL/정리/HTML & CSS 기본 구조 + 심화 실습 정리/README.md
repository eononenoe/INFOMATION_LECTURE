# ✨ HTML & CSS 기초와 심화 실습 총정리

---

# 📌 1. HTML 기본 이해

## 💡 1-1. HTML 기본 구조
- DOCTYPE 선언
- `<html>`, `<head>`, `<body>` 구조 설명

## 💡 1-2. 텍스트 관련 태그
- 제목 태그 `<h1>` ~ `<h6>`
- 문단 태그 `<p>`, 줄바꿈 `<br>`, 수평선 `<hr>`

## 💡 1-3. 리스트 태그
- 순서 없는 리스트 `<ul>`, 순서 있는 리스트 `<ol>`

## 💡 1-4. 링크와 이미지
- 링크 `<a>`, 이미지 `<img>`

# 📌 2. HTML 고급 활용

## 💡 2-1. 폼(Form) 태그 심화
- 회원가입 폼 만들기
- 다양한 input 타입 (text, password, radio, checkbox, select 등)
- 버튼(button, submit)
- 폼 스타일링

## 💡 2-2. 비디오 & 오디오 삽입
- `<video>` 태그 (배경 영상 적용)
- `<iframe>`으로 YouTube 영상 삽입

## 💡 2-3. 데이터 속성(data-*)
- `data-속성` 활용 방법
- 자바스크립트로 dataset 읽어오기
